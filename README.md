# BitVerse - Quantum Civilization Builder

## Overview
BitVerse is a quantum civilization builder with cyberpunk aesthetic. Build your digital empire in this futuristic world where technology and nature merge.

## Features
- Quantum-inspired civilization building mechanics
- Cyberpunk visual theme with neon aesthetics
- Resource management and strategic planning
- Modular architecture for easy expansion

## Getting Started

### Prerequisites
- [Flutter](https://flutter.dev) SDK >= 3.0.0
- [Dart](https://dart.dev) SDK >= 3.0.0

### Installation
```bash
git clone https://github.com/govind/bitverse.git
cd bitverse
flutter pub get
flutter run
```

### Development
```bash
# Run on device or emulator
flutter run -d <device_id>

# Debug mode
flutter run --debug

# Hot reload
flutter run
```
